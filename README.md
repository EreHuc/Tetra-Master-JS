# Tetra-Master-JS
Recreation of FF9 tetra master game in Javascript

# How to

Install dependencies
```bash
npm i
```

Run dev
```bash
npm run dev
```

Build (not working)
````bash
npm run build
````

Linter
```bash
npm run lint
```

Check type
```bash
npm run flow
```
